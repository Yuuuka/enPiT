「このおもちゃが欲しい」、「あのおもちゃが欲しい」と、ママに言っても買ってもらえない...
そんな子供のためにARによるおままごとアプリを開発しました！この"おままごとハウス"があれば、
可愛い家具やおもちゃを遊び放題、たくさん散らかしてもママに怒られない、そう、それがおままごとハウス。
